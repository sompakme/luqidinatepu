First Look At AEW Collision Set

AEW is set to debut a new show, Collision, tonight on TNT. CM Punk will return for the show, and fans are packing Chicago’s All State Arena for the big premiere episode. Now, we have our first look at the new set.

Steve Fall posted a video from inside the venue tonight for Collision. It appears that AEW has spared no expense with their new set as well.

The new set for AEW Collision has quite a presentation. The big Collision logo is also featured above the stage, and big screen surround the entrance as well. Now, we will have to see how it all looks lit up for the big show.

We will have complete coverage of AEW Collision here at Ringside News. Keep checking back for constant updates, because we never know what might happen with Tony Khan’s company.

AEW Collision should be a big show for their premiere episode. Only time will tell if AEW can keep that momentum going.
